# Co-Pilots — Specs

Working title for an asymmetric co-op space shooter built for a dad to play with his 7- and 10-year-old sons. Two players share one ship over the internet — one pilots, one mans the 360° turret.

Built on the same stack as `tetris-battle` and `snakey`: TypeScript + Vite + PeerJS over WebRTC, Canvas 2D, PWA, Vercel.
